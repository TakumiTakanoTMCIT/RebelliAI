# RebelliAI

ロックマンXにインスパイアされた2Dアクションゲームです。

## 概要

- Unity（C#）で制作中のアクションゲームで、プレイヤーの移動やジャンプ、敵との戦闘など基本的なゲームシステムを実装しています。
- 基本的なドット絵をほとんど自分で描いています。

## ゲームクリエイター甲子園 2024 総合大賞ノミネート作品としてノミネートされました
- URL:https://game.creators-guild.com/g4c/35208/

## 現在の実装内容

* 爽快感あるハイスピードアクション
* IntroStageのボス
* 一部アニメーション
